Detailed Description

A serializable container to hold the keys in a query over an index and type of query. Effectively, the QueryKey is a map of column name to predicate value. For example, in the following query:

create table foo (col1 int, col2 int, index (col1,col2));
select * from foo where col1 = 2 and col2 = 5;

the QueryKey would hold the value {col1 -> 2, col2 -> 5} and type EXACT_KEY.
