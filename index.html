What is Honeycomb?

Honeycomb is an open source MySQL storage engine that stores tables into an external data storage backend. With Honeycomb installed, MySQL is able to take advantage of the scalability, fault-tolerance, and tools of an underlying backend while preserving the familiar relational data model and SQL interface. Honeycomb provides a high-level Java interface so that developing additional backends is straightforward, with no need to understand MySQL internals.

Honeycomb currently supports storing tables to HBase and an in-memory store. The HBase backend provides access to the underlying data in offline map/reduce jobs without having to extract or duplicate the data. Existing applications using MySQL as a backend can be transitioned to use HBase with Honeycomb on a table-by-table basis with no changes to application code.
